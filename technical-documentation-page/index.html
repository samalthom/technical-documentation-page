Introduction

HTML stands for HyperText Markup Language and is the code used to structure a web page and its content. It consists of a series elements used to wrap different parts of the content to make it appear or act a certain way. For example, take the following line of content:

Hello, World!

If we want this line to stand by itself, you can specify it is a paragraph by enclosing it in paragraph tags:

<p> Hello, World! </p>
