Introduction

HTML stands for HyperText Markup Language and is the code used to structure a web page and its content. It consists of a series elements used to wrap different parts of the content to make it appear or act a certain way. For example, take the following line of content:

Hello, World!

If we want this line to stand by itself, you can specify it is a paragraph by enclosing it in paragraph tags:

<p> Hello, World! </p>
Anatomy of HTML
Anatomy of HTML

The parts of the above element are as follows:

  1. Opening tag: consisting of the name of the element, wrapped in opening and closing angle brackets.
  2. Closing tag: the same as the opening tag, except it includes a forward slash before the element name.
  3. Content: self explanatory.
  4. Element: the opening tag, closing tag, and the content together comprise the element.

Elements can also have attributes within the opening tag, which contain extra information about the element that you don't want to appear in the actual content. You can nest elements inside other elements too

Here is how HTML individual elements are combined to form an entire HTML page:

<!doctype html> <!--Required preamble-->
<html lang="en-US"> <!--Wraps all the content on the page-->
  <head> <!--Container for all non-content information-->
    <meta charset="utf-8" /> 
    <meta name="viewport" content="width=device-width" />
    <title>My test page</title> <!--The text in the browser tab-->
  </head>
  <body> <!--Contains all content-->
    <img src="images/firefox-icon.png" alt="My test image" />
  </body>
</html>

Images

The image element has no inner content, so is an example of a void element, and embeds an image in the HTML page via the source attribute containing the path to the image file.

There is also an alternative attribute, which allows you to specify descriptive text for users with visual impairs, or if the image does not display correctly.

Marking up Text

Here are some essential HTML elements for marking up text:

  • Headings: specifies that certain parts of your content are headings, or subheadings.
  • Paragraphs: contains paragraphs of texts.
  • Lists: consists of at least two elements, the type of list, and an element for each item instide the list.

    Links

    To add a link, use an anchor tag (<a>):

    • Wrap some text in an achor element.
    • Give the anchor element an href attribute.
    • Fill in the value of this attribute with the web address you want the link to point to.

    Ensure you include the protocol, https://, to avoid unexpected results.
